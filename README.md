# 🌳 Detección de Cambios Forestales — Amazonía Colombiana

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)
![Google Earth Engine](https://img.shields.io/badge/Google%20Earth%20Engine-4285F4?style=flat&logo=google&logoColor=white)

## 📋 Descripción
Detección y cuantificación de cambios en la cobertura forestal 
de la Amazonía colombiana (Caquetá) comparando imágenes 
satelitales Sentinel-2 de 2020 y 2024.

## 🎯 Resultados principales
| Métrica | Valor |
|---|---|
| NDVI promedio 2020 | 0.764 |
| NDVI promedio 2024 | 0.737 |
| Cambio neto NDVI | -0.026 |
| Área con pérdida significativa | 1,285 km² |
| Equivale a | 128,574 canchas de fútbol |
